# Quickstart

This quickstart describes how to generate the book writing plan.

## Prerequisites

- Python 3.11 installed
- The project dependencies installed

## Generating the Plan

To generate the plan, run the following command from the root of the repository:

```bash
python src/cli/main.py generate-plan
```

This will output the plan to the console. To save the plan to a file, you can redirect the output:

```bash
python src/cli/main.py generate-plan > book-plan.md
```
